# 🍔 Smart Restaurant Order Dashboard

A premium, high-impact full-stack Next.js web application designed for modern restaurant management. It features a stunning real-time dark interface to track revenue, manage orders seamlessly, and monitor active tables.

---

## ✨ Features

* **📊 Live Dashboard Architecture:** Monitor total revenue, active table states, and incoming orders at a single glance.
* **🔄 Real-time Order Actions:** Transition order statuses dynamically from 'Pending' to 'Served' with instantaneous UI updates.
* **⚡ Production Ready Serverless Hosting:** Configured for lightning-fast delivery and global availability via Vercel integration.
* **🎨 Modern UI/UX:** Styled with a sleek, dark-mode aesthetic utilizing premium components and visual flow elements.
